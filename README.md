# Ec-Kart - Full Stack E-commerce Application

## Overview

Ec-Kart is a full-stack e-commerce web application designed to provide a seamless online shopping experience.
It includes product browsing, cart management, authentication, and order handling.

---

## Project Structure

This project is divided into two main parts:

* Frontend (User Interface)
* Backend (API & Business Logic)

## How to Run Locally

### Clone Main Repository

```bash
git clone https://github.com/abhishek0112cs221008/Ec-Kart.git
cd Ec-Kart
```

### Initialize Submodules

```bash
git submodule update --init --recursive
```

### Run Backend

```bash
cd backend
mvn spring-boot:run
```

### Run Frontend

```bash
cd frontend
npm install
npm start
```

---

## Environment Variables

Create a `.env` file in the frontend directory:

```env
REACT_APP_API_URL=http://localhost:8080
```
